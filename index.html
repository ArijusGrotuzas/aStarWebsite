This is an A* algorithm example. The example features an iterface where you can specify two points in a grid. The black tiles are obstacles and the white tiles are available tiles.

The algorithm finds the shortest path to arriving to given by maximizing the f heuristic. The f heuristic is determinded by f = g + h, where g is the distance from the current node to starting node, and h is the distance from current node to the goal node.







Map controls

Ratio - ratio of number of black tiles (obstacles) to number of white tiles. Setting this value to low numbers might make the algorithm not work. Needs to be in the range between 1 and 0.

Size - the number of tiles along the y and x axis.
